Quick Start:
============

1. Setup Google Cloud:
   - Get Gemini API key
   - Enable Sheets API
   - Create service account
   - Share Google Sheet

2. Configure Backend:
   cd backend
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env
   # Edit .env with your keys

3. Configure Frontend:
   cd frontend
   npm install
   cp .env.example .env.local

4. Start Everything:
   ./start.sh

5. Open Browser:
   http://localhost:3000
